# Personal Cookbook

This is an Obsidian-based personal cookbook managed in Git.

## How to work in this repo
- Open the repo root as an Obsidian vault.
- Keep recipes in `recipes/`.
- Follow `AGENTS.md` and `reference/style-guide.md`.
- Use `templates/recipe-template.md` for new recipes.

## Goals
- preserve recipes that have actually worked
- maintain strong regional authenticity for Indian food
- store personal notes and iterative improvements
